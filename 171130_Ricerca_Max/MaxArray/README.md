# Ricerca Max
In questo programma si propone la ricerca dell' elemento più grande in un **array** di interi utilizzando un certo numero di **thread** ed un oggetto **CountDownLatch** per la sincronizzazione. Si lavora con la tecnica del **multithreading** quindi programmazione **parallela**. L' array viene **suddiviso** in **n** parti, dove **n** è il numero di **thread**. Ogni thread trova il più grande elemento del proprio **sub-array** e lo salva in un' altro array, che conterrà tutti i massimi valori collezionati dai thread. Infine, quando **tutti** hanno finito, e qui viene usato **CountDownLatch**, si trova quale tra i **massimi** è il **massimo**.
